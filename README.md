## 🎯 Project Overview

This project analyzes user onboarding and conversion performance in a SaaS product.

The goal is to identify drop-off points in the user journey and improve conversion from registration to paid customers.

## 🎯 Business Goal

- Analyze onboarding funnel performance
- Identify conversion bottlenecks
- Improve trial-to-paid conversion rate

## 🧰 Tools Used

- Tableau

## 📈 Key Metrics

- Total registrations
- Trial conversions
- Paying customers
- Conversion rate

## 🔍 Analysis

The funnel includes:

- registration
- email_verification
- profile_completion
- setup_completion
- trial_start
- first_payment

## 💡 Key Insights

- The largest drop-off occurs between trial_start and first_payment
- Onboarding stages perform relatively well
- Monetization step shows the lowest conversion

## 🚀 Business Impact

- Identified critical drop-off stage in monetization
- Provided insights to improve trial-to-paid conversion
- Highlighted need for pricing or UX optimization
